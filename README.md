# Euron MT Game Platform 🎮

A simple web-based game platform made for the Euron MT.

## Features

* Team score tracking
* Multiple mini games

  * 노래 퀴즈
  * 로고 퀴즈
  * 초성 게임
  * 관상 퀴즈
  * 릴레이 그림 그리기
* Answer reveal system
* Projector-friendly UI
* Final leaderboard & winner screen

## Tech Stack

* React / Next.js
* TypeScript
* Tailwind CSS
* localStorage

## Run Locally

```bash
npm install
npm run dev
```

## Deployment

Deploy easily with [Vercel](https://vercel.com?utm_source=chatgpt.com)

## Notes

* This project is designed for offline/team-play MT activities.
* Teams answer on paper while the host controls the platform.
